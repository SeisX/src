These cripts apply basic time procesig to the mobil avo, viking grabem line
12 2D dataset.

basicsu Symes' seismic unix scripts for velocity analysis, stack, and migration.
basicm8r repeat the basicsu velocity analysis and stack using madagascar
fetch -     Madagascar script to download, convery to rsf format, display shots,
            display near trace gather and headerattr.
firstlook - Madagascar scripts to create fold plot, neartrace gather, plot cdp 
            gathers, plot cdp gathers with nmo, velocity analysis, and stack.
Decon     - apply decon, plot cdp gathers with nmo, plot ntg, stack, and 
	    plot stack.
cmp_su_m8r- make plots to compare results in the other directories.
